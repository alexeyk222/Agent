## Генерация через AI-инструменты

### Рекомендуемые инструменты

1. **Midjourney** (лучшее качество, платный)
   - Использовать: `--style expressive --v 6`
   - Для портретов: добавить `--s 750` (stylize)

2. **DALL-E 3** (через ChatGPT Plus)
   - Хорошо понимает русский контекст
   - Можно добавить "in Russian fantasy style"

3. **Stable Diffusion** (бесплатный, локальный)
   - Модель: DreamShaper или Realistic Vision
   - LoRA: add "cozy fantasy illustration"

### Постобработка

После генерации:
1. Убрать артефакты (руки, текст)
2. Цветокоррекция: добавить тепла
3. Slight blur для фонов (фокус на UI)
4. Сжатие для web

## Как использовать промпты

### Для Midjourney:
```
/imagine [промпт из этого файла]
```

### Для DALL-E 3:
```
"Generate an image: [промпт]"
```

### Для Stable Diffusion:
```
Positive: [промпт]
Negative: cold colors, harsh lighting, neon, cyberpunk, photo-realistic, aggressive, scary
Steps: 30-40
CFG: 7-9
```

---

Все промпты готовы к использованию. Можно генерировать батчами или по одному.
